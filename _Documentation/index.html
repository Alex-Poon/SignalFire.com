Installation


Installation is pretty easy.

Unzip the HTML folder, and upload it trough FTP.

Customization


You can custom pretty much everything.


Base Color

The template comes with a default orange base color #FB8047.

To change the base color, open the CSS file located in css/style.css, scroll to the bottom of the file, and in the section "Base color" find the text #FB8047 and replace with your HEX code color.


Logo

There are 2 version of logo: the default (black) version and the white version. This is needed to create contrast with the background.
The logo is present in both blocks: home and contact. By default, the home block has the black version of the logo while the contact block has the white version.

In the /img/ folder you will find 4 logo files:

  • logo.png (Default logo (black version))
  • logo-retina.png (Default logo for retina displays)
  • logo-white.png (White version)
  • logo-white-retina.png (White version for retina displays)
If you want to support High Pixel Density Displays (aka Retina), you just need to have a retina version of it.
A retina version of the logo is basically the exact same logo twice the size. For example, if your logo image is 200x100px you want to create an image with 400x200px.

Just save your logo images with the same name as above and you are good to go.
Make sure you update the HTML file with your logo's dimensions. Search for logo in index.html file and replace accordingly: width="176" height="41"

Content

There's not much content to edit, but you can always add yours. We provide some dummy content to get you started. Feel free to use it or to change it.

The easiest way to change the content, is to open the index.hml file with a visual editor like Dreamweaver.


Contacts

We provide 2 addresses in this theme. However if you want only one address block, it's very easy to remove it.
Find the comment <!-- Begin Address 2 --> and erase or comment everything in between <!-- End Address 2 -->


Social Media

Simply add or remove your social profiles.

Find the block comment <!-- Social Profiles --> and edit or remove the profiles.

Each profile is a <li> element and has a class that identifies the social network.
Just edit the href attribute and replace with your profiles URL


Footer info

Don't forget to change the copyright info.
Find the comment block <!-- Copyright info goes here --> and edit it.


Components


We'll guide you trough the available components.


Timer

There's a timer enabled by default.
You can choose to keep it or not, if you dont want it, open the file js/main.js and find the config settings of the timer.

To disable the timer set the var timer_enabled to false.

To set the future date, in the same config settings you'll find the vars for: year, month and day.


Slideshow

We provide you with a full screen slideshow.
Head to index.html file and find the comment <!-- Begin Slideshow -->.
You can set a video or a gallery of images for the slideshow.

You can't have the slideshow with video and images simultaneously.


Images

By default, the images of slideshow are active. Add or remove the images you want.
If you dont want a slideshow, declare only one image inside the slideshow object.

Video

By default, the video is disabled, but you have an example that's commented.
Uncomment the video block and comment or remove the images inside the slideshow.

You'll need to provide some files in order to get video running:

You should provide a fallback image, just in case the user's browser does not support video. Set this in the poster atribute.
You must provide 3 video formats: mp4, webm, and ogg. Set each in the source parameter.

Next you should set the same settings to the object element and this is set in the parameter flashVars:

{'playlist':['/path/to/poster.jpg',{'url':'/path/to/video.mp4','autoPlay':false}]}

Please note the dummy values in bold.


Twitter

This theme comes with a Twitter feed enabled by default.

If you want to disable it, open the file js/main.js and find the twiter config settings. Set twitter_enabled to false if you want to disable it.

Form

This template comes with an AJAX powered form.
You can use it to send an e-mail, join a newsletter, or anything you want.

The file that's called after submission is form.php. By default this simple PHP script sends you an e-mail everytime there's a new subscription and also writes to a .txt file: emails.txt

Open the form.php file and edit the $to variable with your e-mail. You dont need to change anything if you want the default funcionality.

Edit this file to execute your functions.

AJAX Form
If you want to use third-party services like Mailchimp, Campaign Monitor, you should disable ajax form.
Find the Newsletter form config settings in js/main.js and set the variable ajax_form to false

Map

This theme comes with a awesome fullscreen Google Maps. By default you have 2 points on the map.
Open js/main.js file and find Google Maps config settings.

To disable the Point 2, just change the value of vars google_maps_latitude_2 and google_maps_longitude_2 to null .

Latitude and Longitude values must be valid, otherwise it will crash your site and a kitten may die.

Next to it, you have the color vars, just set the color in HEX for circle points, landscape and water.
